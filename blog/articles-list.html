How to Access Your Articles

To view your published articles:

  1. Save the downloaded HTML file to the blog/articles/ folder
  2. The file will be accessible at: articles/[filename].html
  3. You can then click the "View Article" link to open it
Note: Articles are downloaded as HTML files that you need to manually save to the articles folder.
